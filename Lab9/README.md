# паттерн Заместитель

#### это структурный паттерн проектирования, который подставляет объект-замену вместо реального и перехватывает вызовы, обрабатывая их до обращения к реальному объекту
в лабе применяется оптимизация CoW (Copy-On-Write) - оптимизация, при которой данные не копируются сразу при создании в совместном использовании, а только когда их изменили
